## Recursos

-   Registrar novos Funcionários
-   Registrar Pontos (horários) de funcionários
-   Calcular horas trabalhadas no mês do funcionário


## Instalação

O PyPonto pode ser facilmente inicializado com o comando a seguir:

```shell
docker-compose up
```
